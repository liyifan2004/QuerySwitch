# Icons

This directory should contain the extension icons in the following sizes:
- icon16.png (16x16)
- icon32.png (32x32)
- icon48.png (48x48)
- icon128.png (128x128)

Source file: icon.svg

To generate PNG icons from the SVG, you can use:
- Online converters like https://convertio.co/svg-png/
- ImageMagick: `convert -background none icon.svg -resize 128x128 icon128.png`
- Or any image editing software
